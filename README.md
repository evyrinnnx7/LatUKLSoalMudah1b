Program BiayaPengiriman1 digunakan untuk menghitung total biaya kirim paket berdasarkan berat, jarak, dan ukuran. Program meminta pengguna memasukkan data paket, lalu menghitung volume dan menentukan tarif. Jika jarak ≤10 km tarifnya Rp4.250/kg, jika lebih maka Rp6.000/kg. Paket dengan volume lebih dari 100 cm³ dikenai tambahan Rp50.000. Hasil akhirnya menampilkan rincian biaya dan total ongkir yang harus dibayar.
